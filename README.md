# PharmaX.ai

PharmaX.ai is a lightweight Vite + React + TypeScript application that helps researchers explore drug repurposing opportunities by aggregating market insights, clinical trials, patents, and literature â€” and visualizing relationships with an interactive knowledge graph.

## Quick Features
- Search-driven analysis for a target drug
- Mock dataset for offline/demo use
- Extensible service layer for real data sources (DrugBank, PubChem, PubMed, ClinicalTrials.gov, patents, pathway DBs)
- Interactive knowledge graph visualization
- Dossier export (PDF) functionality
- Component-based UI using shadcn-style primitives

## Getting Started

Prerequisites
- Node.js (18+) and npm

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
